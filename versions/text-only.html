Getting started with HTML and CSS Why learn HTML and CSS? HTML (or HyperText Markup Language) and CSS (or Cascading Style Sheets) are super handy computer languages for making things on the web. Every webpage that you go to using your web browser uses HTML and CSS. HTML is used to "mark up" content (such as this text) and CSS is used to style it. Without either of these, this would look like a big blob of text. HTML basics Here are some basic ideas to help you understand and get started with HTML. HTML is made of elements. Elements are created using tags. You can recognize a tag by its use of triangle brackets: < and > Tags usually (though not always) come in pairs, with the content in between. Both the start and end tags of an element share the same name, however the end tag also includes a forward slash. Most elements can (and often do) contain other elements. Using CSS HTML elements don't have style per se, however most have a default style that is applied if no other style rules exist. In CSS, we use "selectors" to choose what we want to be affected by a rule, and then we provide a list of properties and the values that we want them to be.
